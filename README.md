# plus&fibonacci

plusは数列の足し算をするプログラムです。
fibonacciはフィボナッチ数列を生成するプログラムです。


## インストール方法

以下の手順でプロジェクトをローカル環境にインストールしてください。

$ git clone https://github.com/komugip1000/robosys2024.git

## fibonacci 概要

- フィボナッチ数列を生成するプログラム
- 数字を入力するとその数の項の数列を生成


## fibonacci 使い方

プログラムを実行すると文章が出て

$ ./fibonacci
